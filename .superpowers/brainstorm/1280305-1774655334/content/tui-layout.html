Key Interactions

1

Fuzzy Filter

Type to filter repos instantly (like fzf). Shows all ~/repos dirs, active ones at top.

2

Live Status

Shows which repos have active tmux sessions, whether claude or shell, and remote-control status.

3

Grouped Sections

Active sessions at top, then favourites, then all repos alphabetically. Clear visual separation.

4

Tab Color Coding

TUI tab = orange. Workspace tabs = default. Remote-control tabs = distinct color (red/purple?).
