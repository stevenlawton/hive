TUI controls via:
kitty @
• launch (new tab)
• set-tab-title
• set-tab-color
• focus-tab
• close-tab
• ls (query state)
tmux
• new-session -s kl-<name>
• send-keys (launch claude)
• has-session (check alive)
• kill-session (cleanup)
• list-sessions (reconnect)
Session Lifecycle
1. User hits Enter on "SliceWise" in TUI
2. TUI runs: tmux new-session -d -s kl-slicewise -c ~/repos/SliceWise
3. TUI runs: tmux send-keys -t kl-slicewise 'claude' Enter
4. TUI runs: kitty @ launch --type=tab --title="SliceWize" tmux attach -t kl-slicewise
5. TUI runs: kitty @ set-tab-color -m title:SliceWize active_bg=#ff6b6b
On kitty restart → TUI detects existing tmux sessions (kl-* prefix), re-attaches tabs
