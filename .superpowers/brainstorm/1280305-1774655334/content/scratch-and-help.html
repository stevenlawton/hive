Scratch cleanup

A

Manual only

Scratch dirs live in /tmp so they disappear on reboot. TUI shows them while they exist, that's it.

B

Prompt on quit

When quitting TUI, if unpromoted scratches exist, ask "Promote or discard?" for each.

C

Age-based warning

If a scratch is >24h old, show a warning badge in the TUI. Nudge to promote or kill, but don't auto-delete.
